
# FAQ Chatbot using NLP

## Description
This project is a chatbot designed to answer frequently asked questions.
It uses NLP techniques like text preprocessing, TF-IDF vectorization, and
cosine similarity to match user queries with the most relevant FAQ.

## Features
- FAQ-based chatbot
- NLP preprocessing using NLTK
- Cosine similarity for intent matching
- Optional chat UI using Tkinter

## Technologies
- Python
- NLTK
- scikit-learn

## Run
pip install -r requirements.txt  
python chatbot.py
