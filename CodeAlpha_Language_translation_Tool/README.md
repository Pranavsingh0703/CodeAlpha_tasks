# Language Translation Tool

## Description
A simple language translation tool built using Python and Tkinter. 
It allows users to enter text, select source and target languages, 
and translate text using Google Translate API.

## Features
- Text input with language selection
- Real-time translation
- Copy translated text to clipboard

## Technologies
- Python
- Tkinter
- Googletrans API

## Run
pip install -r requirements.txt  
python main.py
